CallStrike is an AI-powered outbound calling agent that autonomously calls any phone number, conducts a real voice conversation using a custom prompt you define, and returns a clean summary of what the other person said.
Built with Gemini for conversation intelligence, Groq for fast inference, and Twilio for real-world phone calls. Fully bring-your-own-credentials — your API keys, your calls, your data. No keys are stored server-side.
Built as a portfolio project by a CSE student exploring the intersection of voice AI and real-world automation.
